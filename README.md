# AI Meme & Poster Creator

An AI-powered web application that generates memes and posters using templates and AI-generated captions.

## Features
- AI caption generation
- Meme creation
- Poster generation
- Image upload
- Download output

## Technologies
- Python
- Streamlit
- Pillow
- Hugging Face / Gemini API
